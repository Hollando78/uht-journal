## Observation

Two distinct ecological concepts — ecological niche and trophic cascade — produced identical hex codes (00048200), activating the same three traits: System-Integrated, Symbolic, and Temporal. A second collision appeared between food web and predator-prey dynamics, both mapping to 00059200 with five shared traits. Meanwhile, carrying capacity, a fundamentally quantitative concept describing maximum sustainable population, collapsed to a single active trait (Symbolic) at 00008000 — one of the sparsest classifications seen across any domain.

These collisions are not trivial overlaps between near-synonyms. Ecological niche describes a static relational position; trophic cascade describes a dynamic propagation effect. That UHT assigns them the same code suggests the current trait set cannot distinguish between "occupying a role within a system" and "propagating effects across a system." Similarly, food web (a structural map) and predator-prey dynamics (a temporal process) collapse together, indicating UHT conflates network structure with the dynamics that play out on that structure.

## Interpretation

Ecology stresses the trait set differently than prior domains. The structure-vs-dynamics conflation (niche/cascade, web/dynamics) reveals a missing discriminator. A trait like "Relational-Structural" (describing static network position) versus "Causal-Propagating" (describing dynamic effects traversing a structure) would separate these pairs. Carrying capacity's sparse classification suggests UHT lacks a trait for quantitative boundary concepts — limits, thresholds, capacities. The existing trait set captures what something *does* in a system but not what *constrains* that system.

Keystone species' bridge to both biological and ethical/existential entities is genuinely interesting — it suggests UHT recognizes that "disproportionate influence" is a cross-domain archetype, shared between ecological keystones and concepts humans mark as existentially significant.
